# Bamboo Diary

简单日记应用部署在新浪云上.
网址:http://bambooomdiary.sinaapp.com/

* 网页端
	* 打开http://bambooomdiary.sinaapp.com/
	* 可以输入新的吐槽及标签
	* 将在下方显示历史记录
* CLI端
	* 预先安装Requests以及BeautifulSoup
	* 运行```client.py```
	* 输入``st:<TAG>```添加新标签并输入新日记
	* 输入```s/sync```获取当前标签下的所有日记
	* 输入```lt/ListTags```获取所有标签列表
	* 输入```FLUSH```清楚所有日记
